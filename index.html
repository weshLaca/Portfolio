N.01 Demo live

Neon Drift

Endless runner arcade, canvas puro.

Un runner arcade in HTML5 Canvas: schiva gli ostacoli, raccogli i core per ricaricare l'energia e attiva la Phase per attraversarli in un attimo. Loop di gioco a requestAnimationFrame, particelle, trail dinamici e un motore audio scritto da zero con la Web Audio API — nessun file mp3, solo oscillatori.

HTML5 Canvas JavaScript Web Audio API
